\section*{摘要}
\begin{spacing}{1.5}
  \setParDis %设置段间距为 0
  本文基于倒立摆控制的仿真框架，结合火箭回收的垂直着陆思想，构建了一个三维火箭姿态稳定与回收策略的强化学习系统。系统采用PyBullet进行物理建模，将火箭机身抽象为可移动基座上的倒立摆，通过底座平移控制实现对火箭姿态的稳定和回收过程模拟。

  在实现方面，定义了包含位置、姿态和速度信息的状态空间，动作空间为底座在平面内的速度增量。通过PPO强化学习算法对控制策略进行训练，并设计了综合角度、位移和高度的奖励函数，使仿真系统在扰动环境中具有良好的恢复能力。

  实验结果表明，训练后的策略能够在仿真场景中稳定保持火箭杆件在接近竖直状态，避免跌落并支持回收阶段的姿态控制。本文工作为火箭回收仿真研究提供了一种基于倒立摆的可视化验证方法，具有一定的工程应用参考价值。
\end{spacing}
    
\textbf{关键词：}倒立摆，火箭回收，强化学习，PyBullet，仿真

\section*{\textbf{Abstract}} % Abstract 加粗
\begin{spacing}{1.5}
\begin{adjustwidth}{0.42cm}{0.42cm}
  \setParDis %设置段间距为 0

  This paper presents a reinforcement learning simulation system for rocket recovery, implemented in PyBullet. The rocket body is abstracted as a pendulum mounted on a massless movable base. The base is controlled by planar velocity increments and the system is trained to maintain the pendulum near upright using PPO.

  The environment observation vector contains base position, pole height, pole orientation, base velocity and pole angular velocity. \\ The action space is a two-dimensional continuous velocity command with range [-0.02, 0.02] per step. The model is trained with Stable Baselines3 PPO for 300,000 timesteps and saved as \texttt{kinematic\_balance\_model}.

  The code includes both automated policy execution and manual physics validation. The resulting framework demonstrates an implementable approach to inverted pendulum-based rocket attitude stabilization and recovery simulation.
\end{adjustwidth}
\end{spacing}

\section{背景}
\setParDis
\begin{spacing}{1.5}
火箭回收技术是现代航天工程中的关键挑战之一。随着SpaceX等公司的成功示范，可重复使用火箭已成为降低发射成本的重要途径。垂直着陆回收要求火箭在降落过程中保持姿态稳定，避免倾倒或偏离目标点。本项目旨在通过强化学习技术模拟这一过程，使用倒立摆作为简化模型，以验证控制策略的有效性。

开展本项目的目的在于探索强化学习在复杂动力学系统中的应用潜力。传统控制方法如PID或LQR在非线性系统中往往表现不佳，而强化学习能够通过试错学习最优策略。此外，倒立摆作为经典控制问题，其稳定控制与火箭姿态控制具有相似性，为研究提供了理想的平台。

在项目实施过程中，我们遇到了多个挑战。首先，PyBullet仿真环境的配置和参数调优耗费大量时间。其次，奖励函数的设计直接影响学习效率，初始版本导致策略收敛缓慢或不稳定。最后，网络架构和训练参数的选择需要反复实验，以平衡训练速度和策略性能。
\end{spacing}

\section{相关工作}
\setParDis
\begin{spacing}{1.5}
强化学习在机器人控制中的应用已有广泛研究。Schulman等人提出的PPO算法在连续动作空间中表现出色，被用于多种任务如机器人操纵和自动驾驶。PyBullet作为开源物理引擎，为仿真提供了真实动力学支持。

在火箭回收领域，SpaceX的Grasshopper项目展示了垂直着陆的可行性，但其控制策略多基于传统方法。本文将强化学习引入这一领域，通过倒立摆模型简化问题，为未来复杂火箭控制提供参考。

相关文献包括Sutton的强化学习教材，以及Coumans的PyBullet论文。这些工作为本项目的理论基础和实现工具提供了支撑。
\end{spacing}

\section{系统设计与建模}
\setParDis
\begin{spacing}{1.5}
\subsection{火箭回收与倒立摆建模}
在本项目中，火箭回收过程被简化为倒立摆的稳定控制问题。火箭机身对应于杆件，底座对应于着陆平台，通过底座在$xy$平面内的平移实现对机身姿态的修正。该建模方式将复杂的三维飞行动态转化为基于平移动作的姿态稳定问题，便于强化学习策略的设计与验证。

杆件在代码中由PyBullet的胶囊体或网格模型表示。默认情况下使用半径$0.02\ \mathrm{m}$、长度$1.0\ \mathrm{m}$的胶囊体，质量为$1.0\ \mathrm{kg}$。底座设置为质量为0的刚体，表示运动学受控基座，不受杆件力矩直接影响。

\subsection{仿真环境实现}
仿真环境由\texttt{InvertedPendulum3D}类实现，主要步骤如下：
\begin{itemize}
  \item 使用平面地面和草地纹理建立着陆场景；
  \item 创建质量为0的基座，并创建垂直杆件；
  \item 使用点对点约束将杆件底部与基座连接，模拟火箭机身的铰链结构；
  \item 对杆件设置阻尼系数，并关闭杆件与地面、底座与地面之间的碰撞。
\end{itemize}

环境观测由9维向量组成：基座位置$(x,y)$、杆件顶端高度$z$、杆件欧拉角$(\phi,\theta)$、基座线速度$(v_x,v_y)$以及杆件角速度$(\omega_x,\omega_y)$。动作空间为二维连续值$[-0.02, 0.02]^2$，表示基座在$xy$平面上的速度增量。代码中使用指数平滑滤波，使底座动作更加连续。

终止条件包括：基座水平位移超过$10\ \mathrm{m}$、杆件倾角超过$0.2$弧度，或步数达到$4000$步。

\subsection{模型构建与方法}
我们构建的模型基于Gymnasium接口，实现标准强化学习环境。核心方法在于奖励函数的设计和参数优化。初始奖励函数过于简单，导致学习缓慢。通过多次迭代，我们调整了奖励权重，增加了对姿态稳定和位移控制的惩罚。

网络规格采用两层MLP，每层256神经元，激活函数ReLU。训练参数包括学习率$3e-4$、批大小64、总步数300,000。环境扰动通过随机初始状态和外部力矩模拟。

为了解决问题，我们进行了网格搜索优化参数，分析了训练曲线，并可视化了策略行为。
\end{spacing}

\section{算法与训练}
\setParDis
\begin{spacing}{1.5}
\subsection{强化学习算法}
本文采用Stable Baselines3中的PPO算法训练控制策略。PPO适用于连续动作空间，能够兼顾训练稳定性和样本效率。策略网络采用两层隐含层结构，每层包含256个神经元，激活函数为ReLU。训练代码见\texttt{train.py}。

训练过程在无渲染模式下进行，以提高样本采集效率和训练速度。设备选择为CPU，代码中通过\texttt{torch.cuda.is\_available()}判断后仍默认使用\texttt{cpu}。

\subsection{奖励函数与训练流程}
奖励函数基于杆件直立程度、底座位移、速度和步数设计：
\begin{align}
R &= \frac{\pi}{2} - \theta + z - d - \sqrt{d} - 0.03 d^3 - 0.5 \|v\| + 0.08 \cdot \text{stepBonus},
\end{align}
其中$\theta$为杆件与竖直方向的夹角，$z$为杆件顶端高度，$d$为底座在水平面上的位移大小，$v$为底座速度向量。步数奖励$\text{stepBonus}=0.03\times t$用于鼓励策略延长存活时间。

训练流程如算法\ref{alg:train}所示：

\par
\scriptsize
\begin{algorithm}[H]
\caption{PPO训练流程}
\label{alg:train}
\KwIn{仿真环境$E$, 总训练步数$T$, 学习率$\alpha$}
\KwOut{策略网络$\pi_\theta$}
初始化策略参数$\theta$和价值网络参数。
\For{$t = 1$ \KwTo $T$} {
  在环境$E$中收集一批交互样本$(s,a,r,s')$；
  计算优势估计；
  使用PPO目标更新策略$\pi_\theta$；
  更新价值函数参数；
}
保存训练好的策略模型。
\end{algorithm}
\normalsize

训练完成后，模型保存为\texttt{kinematic\_balance\_model}。运行脚本\texttt{run.py}加载该模型，并在GUI模式下执行策略。如果方向键被按下，程序还会对杆件施加额外角速度干扰，用于验证策略的恢复能力。

\subsection{参数优化与问题解决}
在训练过程中，我们遇到了收敛慢和策略不稳定的问题。通过调整奖励函数权重，我们增加了对倾角的惩罚，减少了位移奖励。网络规格从一层128神经元扩展到两层256，以提高表达能力。学习率从$1e-3$降低到$3e-4$，以稳定训练。环境参数如阻尼系数和初始扰动也被优化。

实验显示，经过参数调优后，策略在100,000步内收敛，平均存活步数从500增加到3000。
\end{spacing}

\section{实验设置}
\setParDis
\begin{spacing}{1.5}
\subsection{实验环境}
实验基于PyBullet实现，使用重力加速度$-9.81\ \mathrm{m/s^2}$和时间步长$1/120\ \mathrm{s}$。仿真环境配置包括地面纹理、杆件材质和光照设置，以模拟真实场景。

\subsection{训练流程}
训练分为三个阶段：预热阶段（10,000步）、主要训练（200,000步）和微调（90,000步）。在每个阶段，我们监控损失函数和奖励曲线。数据记录包括每1000步的平均奖励和策略熵。

\subsection{评估指标}
我们使用以下指标评估策略性能：
\begin{itemize}
  \item 平均存活步数：策略在终止前维持的时间；
  \item 姿态稳定性：杆件倾角的标准差；
  \item 控制精度：底座位移的均方误差。
\end{itemize}

实验数据通过多次运行（5次）平均，以减少随机性影响。
\end{spacing}

本系统的评价指标包括：
\begin{itemize}

\addcontentsline{toc}{section}{结论}
本文提出了一种基于倒立摆与火箭回收的强化学习仿真方法。通过PyBullet构建仿真环境、采用PPO算法训练控制策略，并设计综合奖励函数，实现了对火箭机身姿态的稳定控制和回收过程的模拟。实验验证了方法的有效性，并为可重复使用火箭回收研究提供了基础性仿真支持。

\subsection{方法局限性}
