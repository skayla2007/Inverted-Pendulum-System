\section*{摘要}
\begin{spacing}{1.5}
  \setParDis %设置段间距为 0
  本文基于倒立摆控制的仿真框架，结合火箭回收的垂直着陆思想，构建了一个三维火箭姿态稳定与回收策略的强化学习系统。系统采用PyBullet进行物理建模，将火箭机身抽象为可移动基座上的倒立摆，通过底座平移控制实现对火箭姿态的稳定和回收过程模拟。

  在实现方面，定义了包含位置、姿态和速度信息的状态空间，动作空间为底座在平面内的速度增量。通过PPO强化学习算法对控制策略进行训练，并设计了综合角度、位移和高度的奖励函数，使仿真系统在扰动环境中具有良好的恢复能力。

  实验结果表明，训练后的策略能够在仿真场景中稳定保持火箭杆件在接近竖直状态，避免跌落并支持回收阶段的姿态控制。本文工作为火箭回收仿真研究提供了一种基于倒立摆的可视化验证方法，具有一定的工程应用参考价值。
\end{spacing}
    
\textbf{关键词：}倒立摆，火箭回收，强化学习，PyBullet，仿真

\section*{\textbf{Abstract}} % Abstract 加粗
\begin{spacing}{1.5}
\begin{adjustwidth}{0.42cm}{0.42cm}
  \setParDis %设置段间距为 0

  This paper presents a reinforcement learning simulation system that integrates inverted pendulum control with rocket recovery for vertical landing. The physical model is implemented in PyBullet, where the rocket body is abstracted as a pendulum mounted on a movable base. The base is driven by planar velocity actions to stabilize the rocket attitude and emulate the recovery sequence.

  The environment encodes state observations including position, orientation and velocity, while the action space represents planar base velocity increments. A PPO policy is trained using a reward function that balances pole angle, height and displacement, enabling robust recovery behavior under disturbance.

  Experimental results demonstrate that the learned policy can keep the rocket-like pendulum near the upright state, avoid failure, and support simulated recovery behavior. This study introduces a practical inverted pendulum framework for rocket recovery simulation and serves as a reference for future vertical landing research.
\end{adjustwidth}
\end{spacing}

\section{引言}
\setParDis
\begin{spacing}{1.5}
本项目以倒立摆控制为基础，探索火箭回收垂直着陆过程的仿真与策略学习。火箭回收的关键在于在落地前保持机身姿态稳定并精确对准降落点，这与倒立摆的稳定控制问题具有高度相似性。本文通过建立三维倒立摆仿真环境，并结合强化学习算法，实现了对火箭姿态恢复与回收过程的可视化验证。

火箭回收技术是现代航天发展的重要方向，尤其是在可重复使用火箭的垂直着陆阶段。利用倒立摆模型模拟火箭机身，将其视为可在平面内移动的基座之上的杆件，使得控制设计更加直观。本文的贡献包括：
\begin{itemize}
  \item 构建基于PyBullet的三维倒立摆仿真环境，将火箭模型抽象为可移动基座上的倒立摆结构；
  \item 使用PPO强化学习算法训练控制策略，设计综合角度、位移、高度的奖励函数，提高系统稳定性；
  \item 结合火箭回收概念，展示了基于倒立摆的姿态恢复方法在垂直着陆仿真中的应用价值。
\end{itemize}
\end{spacing}

\section{系统设计与建模}
\setParDis
\begin{spacing}{1.5}
\subsection{火箭回收与倒立摆建模}
在本项目中，火箭回收过程被简化为倒立摆的稳定控制问题。火箭机身对应于倒立摆杆件，底座对应于着陆平台或控制表面，通过底座在$xy$平面内的平移实现对机身姿态的修正。该建模方式将复杂的三维飞行动态转化为基于平移动作的姿态稳定问题，便于设计强化学习策略。

杆件采用长度为$L$、质量为$m$的竖直杆模型。状态量包括杆件顶端位置和欧拉角，动作量为底座速度增量$[v_x, v_y]$。这种简化模型保留了火箭回收时对机身姿态控制的核心特性，并可通过仿真验证策略鲁棒性。

\subsection{仿真环境实现}
仿真环境基于PyBullet构建，主要实现步骤如下：
\begin{itemize}
  \item 使用平面地面和草地纹理建立着陆场景；
  \item 创建质量为0的基座和杆件，杆件可采用胶囊体或火箭网格模型；
  \item 通过点对点约束使杆件底部与基座连接，模拟火箭机身的铰链结构；
  \item 设置重力、动力学参数和碰撞过滤，确保仿真稳定。
\end{itemize}

状态观测包括：杆件顶端位置、杆件高度、欧拉角、基座速度和角速度。动作空间为$[-\Delta v, \Delta v]$区间内的二维连续速度控制。终止条件包括杆件倾角过大、基座偏移过远或达到最大步数。
\end{spacing}

\section{算法与训练}
\setParDis
\begin{spacing}{1.5}
\subsection{强化学习算法}
本文采用Stable Baselines3中的PPO算法训练控制策略。PPO具有收敛稳定、样本效率较高的优点，适合处理本项目中的连续动作空间。策略网络采用两层隐含层结构，每层包含256个神经元，激活函数为ReLU。

训练过程中使用的关键超参数包括：学习率$3\times10^{-4}$、折扣因子$\gamma=0.99$、步长$2048$、批量大小$128$。训练过程在无渲染模式下进行，以提高训练效率。

\subsection{奖励函数与训练流程}
奖励函数设计是控制策略成功的核心。本文采用以下奖励项组合：
\begin{align}
R &= w_1 \left(\frac{\pi}{2} - \theta\right) + w_2 z - w_3 d - w_4 \sqrt{d},
\end{align}
其中$\theta$为杆件与竖直方向的夹角，$z$为杆件顶端高度，$d$为基座在水平面上的位移大小。通过增强$z$和减小$\theta$，策略能够更好地保持火箭姿态直立并控制着陆点偏移。

训练流程如算法\ref{alg:train}所示：
\begin{algorithm}[H]
\caption{PPO训练流程}
\label{alg:train}
\KwIn{仿真环境$E$, 总训练步数$T$, 学习率$\alpha$}
\KwOut{策略网络$\pi_\theta$}
初始化策略参数$\theta$和价值网络参数。
\For{$t = 1$ \KwTo $T$} {
  在环境$E$中收集一批交互样本$(s,a,r,s')$；
  计算优势估计；
  使用PPO目标更新策略$\pi_\theta$；
  更新价值函数参数；
}
保存训练好的策略模型。
\end{algorithm}

训练后，模型保存为"kinematic_balance_model"，可用于后续的仿真运行与可视化展示。
\end{spacing}

\subsection{结果分析}
训练结果显示，PPO策略能够使火箭杆件在扰动环境中快速恢复至接近竖直状态。基座位移得到有效压制，杆件顶端高度维持在稳定区间。该结果验证了倒立摆模型在火箭回收仿真中的可行性，并说明强化学习可以作为复杂着陆控制策略的候选方法。

同时，仿真环境的可视化效果有助于直观展示火箭回收过程，便于进一步优化奖励函数和引入更多外界扰动。未来可在该框架基础上加入风载荷、推力矢量控制等更接近真实火箭回收的因素。
\end{spacing}

\section*{结论}% section*生成无标号章节
\addcontentsline{toc}{section}{结论}
本文提出了一种基于倒立摆与火箭回收的强化学习仿真方法。通过PyBullet构建仿真环境、采用PPO算法训练控制策略，并设计综合奖励函数，实现了对火箭机身姿态的稳定控制和回收过程的模拟。实验验证了方法的有效性，并为可重复使用火箭回收研究提供了基础性仿真支持。

未来工作将进一步扩展该模型，加入更复杂的动力学、推力控制和环境扰动，以提升仿真结果的工程参考价值。
